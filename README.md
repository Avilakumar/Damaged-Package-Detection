# Damaged Package Detection using CNN and MobileNetV2

## Overview
This project detects whether a package is damaged or intact using Deep Learning.

## Models Used
- Custom CNN
- MobileNetV2 (Transfer Learning)

## Dataset
Damaged and Intact Packages Dataset

## Features
- Image preprocessing
- Data augmentation
- CNN model
- MobileNetV2 model
- Test split evaluation
- Confusion Matrix
- Classification Report
- Grad-CAM Visualization

## Results
Model performance comparison using Accuracy and Loss.

## Technologies
Python, TensorFlow, OpenCV, NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn
